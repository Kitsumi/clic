#CLIc
A cli chat I made.

##Install (server)
```
npm install
```

##Usage (server)
```
node server.js <port>
```
If no port is specified, it will use `1111`.

##Install (client)
You can install from git, or use npm
####Npm:
`sudo npm install --global clic-client`
####Git:
`sudo npm install --global`

##Usage (client)
```
clic <ip> <port>
```
If no port is specified, it will use `1111`.
